This page demonstrates the AP Classroom Assistant Chrome extension functionality and provides installation instructions.

✨ Key Features

🔐
Session Keep-Alive

Prevents automatic logout from College Board

📋
Assignment Navigation

Quick access to all active assignments

🚀 Installation Instructions

Open Chrome and navigate to chrome://extensions/
Enable "Developer mode" in the top right corner
Click "Load unpacked" button
Select the AP-Classroom-Plugin folder
The extension icon will appear in your toolbar

🔧 Technical Implementation

The extension uses modern Chrome Extension APIs:

Background Service Worker

// Session keep-alive every 10 minutes chrome.alarms.create('sessionKeepAlive', { delayInMinutes: 1, periodInMinutes: 10 }); chrome.alarms.onAlarm.addListener((alarm) => { if (alarm.name === 'sessionKeepAlive') { keepSessionAlive(); } });

Content Script Assignment Extraction

// Extract assignments from AP Classroom pages const assignmentLinks = document.querySelectorAll('a[href*="/assignment/"]'); assignmentLinks.forEach(link => { const assignment = extractAssignmentInfo(link); if (assignment && assignment.status === 'active') { assignments.push(assignment); } });

🧪 Testing

To test the extension:

  1. Install the extension using the instructions above
  2. Log in to College Board at https://collegeboard.org
  3. Visit AP Classroom at https://apstudents.collegeboard.org
  4. Navigate through your courses to let the extension collect data
  5. Click the extension icon to see your assignments

Debug Mode: Go to chrome://extensions/ and click "Inspect views: background page" to see console logs.

⚠️ Important Notes

  • Requires College Board Account: You need an actual College Board account with AP Classroom access for full functionality
  • Privacy First: The extension stores data locally only and never transmits your credentials
  • Not Official: This is an independent tool, not affiliated with College Board
  • Academic Integrity: Use responsibly and follow your school's technology policies

📞 Support

If you encounter issues:

  • Check the browser console for error messages
  • Verify you're logged in to College Board
  • Make sure you've visited your AP Classroom pages
  • Try the refresh button in the extension popup
